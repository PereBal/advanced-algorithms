## Notes
Pyqtgraph is installed locally because it's pip version does not support
PyQt5. To do so:
```sh
# Get the repo
git clone https://github.com/pyqtgraph/pyqtgraph /tmp

# Store the package
cp /tmp/pyqtgraph/pyqtgraph .

# Remove the repo
rm -fr /tmp/pyqtgraph
```
Numpy is a dependence of it
